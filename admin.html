REST-like Product API

Reference for all available catalog operations.

POST /products

Create a new product entry in the catalog using the form on this page.

GET /products

Fetch the current catalog items from browser storage. Used by shop, homepage, and product pages.

PUT /products/{id}

Edit an existing product. Select one from the table and save changes.

DELETE /products/{id}

Remove a product from the catalog instantly with the delete button.

Note: Because this site is static and has no server-side backend, changes are stored only in the browser. Use the export/import buttons to move catalog data between browsers or to commit updates manually to your GitHub Pages repository.
